## 一、澳門政府對教育科技Startup的補助計劃

### 1. 青年創業援助計劃（經濟及科技發展局）

- **主管部門**：經濟及科技發展局（DSEDT）工商業支援處
- **宗旨**：為擁有創業理想但缺乏資源的本地青年提供免息援助款項
- **援助金額上限**：澳門幣 **30萬元**
- **最長還款期**：**10年**（免息）
- **申請資格**：
  - 年齡：21至44歲
  - 須為澳門永久性居民
  - 企業在澳門營運**不超過兩年**
  - 企業須已在財政局登記
  - 須完成不少於**42小時創業培訓課程**（由公立/私立高等教育機構或澳門生產力暨科技轉移中心舉辦）
  - 具有工商管理學位或同等學歷者可豁免培訓要求
- **援助款項用途**（限於以下用途）：
  - 購置商業企業營運所需的設備
  - 為商業企業營運場所進行裝修工程
  - 訂立商業特許合同或特許經營合同
  - 取得技術專用權或知識產權
  - 進行宣傳及推廣活動
  - 作為商業企業的營運資金
- **申請地點**：
  - 經濟及科技發展局（澳門南灣羅保博士街1-3號2樓）
  - 政府綜合服務大樓（黑沙環新街52號）
  - 離島政府綜合服務中心（氹仔哥英布拉街225號3樓）

### 2. 科學技術發展基金（FDCT）

- **主管部門**：經濟及科技發展局
- **資助計劃（2025年度）**包括：
  1. **科研平台資助計劃**
  2. **科研及創新資助計劃**（A類及B類）
  3. **企業創新研發資助計劃**（可與商業企業合作，企業須提供不低於資助金額10%的配套投入）
  4. **科技研發人才資助計劃**
  5. **科學技術獎勵計劃**
  6. **對外科技合作資助計劃**（聯合資助類，與國家科技部聯合資助）
- **優先支持範疇**（按《澳門特別行政區經濟和社會發展第二個五年規劃》）：
  - 普及和深化科技知識的項目
  - 有助企業提高生產力和加強競爭力的項目
  - 有助產業發展的研發、推廣、創新項目
  - 有助推動研發成果轉化的科研項目
  - 有助推動對外科技合作的項目
  - 推動對社會經濟發展屬優先的科技轉移的項目
  - 專利申請
- **產學研配對資助**：設有平台及資助計劃，促進產學研融合
- **與商業企業合作規定**：若與商業企業合作，企業須提供配套投入（不低於科技基金資助金額10%）

### 3. 教育基金

- **資助範圍**：主要資助大專獎助學金、高等院校年度資助計劃
- **赴葡升學助學金**：由教育及青年發展局管理
- **注意**：教育基金主要針對學生資助，而非直接資助教育科技Startup

### 4. 工商業發展基金的其他支援

- **中小企業援助計劃**
- **中小企業信用保證計劃**
- **中小企業專項信用保證計劃**
- **鼓勵企業升級發展補貼計劃**

### 5. 稅務鼓勵措施（經濟及科技發展局）

- **工業政策範圍內稅務鼓勵**（PS-1334）
- **從事科技創新業務企業的稅務優惠制度**（由財政局負責）

## 二、澳門大學/科大的創業孵化器

### 澳門大學（UM）創新創業中心（CIE - Centre for Innovation and Entrepreneurship）

- **成立時間**：2017年
- **認證**：2019年獲批為**「國家級眾創空間」**（港澳地區首個）
- **定位**：澳大「五位一體」研究創新及轉化體系的重要組成部分
- **孵化成果**：已成功孵化**超過50間初創企業**，涵蓋文化創意、環保、新材料、信息科技、醫療及生物科技等多個領域
- **服務對象**：澳大師生、校友及教職員工
- **支援服務**：
  - 辦公空間（澳門氹仔大學大馬路澳門大學聚賢樓N1 1樓1021室）
  - 培訓指導
  - 宣傳推廣
  - 法律財稅及商業顧問
  - 資源對接
- **入駐申請**：每年一月和八月開放申請
- **業務範疇涵蓋**：醫療科技、資訊及通訊科技、新材料、**教育及旅遊**等
- **產學研合作夥伴**：
  - 廣東大智匯控股集團有限公司
  - 珠海大橫琴發展有限公司
  - 滬港中科國際生物科技有限公司
  - 大灣區知名高校創業孵化平台

### UMTEC（澳門大學產學研合作平台）

- **職能**：
  - 專業研究、技術創新、技術諮詢、技術顧問、技術轉移
  - 協辦學術會議
  - 擔任澳大與產業界之間的橋樑
  - 橫向研發合作、技術落地和專利授權
- **特別職能**：
  - 代表澳大持有由澳大創新創業中心孵化的初創公司股份
  - 協助澳大在內地設立研發機構
  - 開展跨境產學研合作

### 澳門青年創業孵化中心（MYEIC）

- **成立時間**：2015年（隸屬經濟局）
- **2018年**：獲國家科技部授予**港澳地區首個「國家級眾創空間」**
- **2025年5月1日**：遷往南灣新址
- **營運**：由澳中致遠投資發展有限公司營運（全天候運作）
- **面積**：1300平方米
- **服務內容**：
  - 創業培訓課程資訊
  - 諮詢服務
  - 設立公司一站式服務
  - **免費臨時辦公地點**
  - 創業顧問輔導服務
  - **商業配對服務**
  - 經驗交流會
  - 協助青年參與區域經濟合作發展
  - 路演廳（可容納60人）
- **品牌活動**：
  - 澳門青年創新創業大賽
  - 巴葡科技企業（澳門）創新創業大賽
  - 葡語國家科創企業參訪粵港澳大灣區考察團

### 澳門科技大學（MUST）

- **研究領域**：集成電路、生物醫藥等
- **設有研究基金項目（FRG）**：資助範圍包括文理各科研究領域的原創性研究項目
- **科研平台**：擁有全國重點實驗室等創新平台
- **創業指引**：提供創業資訊，指引青年善用政府及大學資源

## 三、澳門人去香港拓展業務的可能性

### CEPA框架

- **澳門與內地的CEPA**：自2004年起全面實施
- **澳門服務提供者證明書**：累計發出超過429張
- **零關稅貨物累計出口金額**：截至2012年約3.54億澳門元

### 澳門 vs 香港的CEPA差異

- 內地對澳門開放的273種產品中，19種為食品
- 對澳門開放的電子產品較多為成品（如電飯煲、電熨斗）
- 對香港開放的較多為配件（與兩地製造業結構相適應）

### 香港拓展的限制

- **CEPA主要針對澳門→內地**，而非澳門→香港的直接優惠
- 內地對香港投資者在外商投資管理體制下須按《外商投資商業領域管理辦法》執行
- 澳門創業者若要進入香港市場，**沒有專門的CEPA-type優惠**

### 可行策略

1. **先在澳門建立基礎，再借大灣區框架進軍香港**
2. **利用香港國際化平台**：作為金融、貿易、仲裁中心
3. **橫琴粵澳深度合作區**：作為澳門→香港/國際市場的跳板
4. **香港科技園/數碼港**：可考慮在港設立分支機構
5. **粵港澳大灣區合作**：善用區域一體化政策

## 四、澳門初創企業的稅務環境

### 整體税制特點

- **低稅率、簡單稅制**
- **屬於免稅港地區之一**
- **實施收入來源地稅收管轄權原則**（非全球徵稅）
- **稅種少、稅負輕**

### 主要稅種

| 稅種 | 說明 |
|------|------|
| **所得補充稅** | 企業所得稅（純利稅），是主要企業稅種 |
| **營業稅** | 幾乎所有場所均免納 |
| **職業稅** | 個人所得稅 |
| **房屋稅** | 不動產稅 |
| **印花稅** | 各類交易文件 |
| **房屋稅** | 租賃收益徵收 |

### 所得補充稅（企業所得稅）

- **徵稅對象**：在澳門從事工商業活動取得的利潤
- **稅率**：**12%**（單一税率）
- **免稅額（2024年度）**：**澳門元60萬元**
  - 可課稅收益首60萬元免稅
  - 超過60萬元的收益適用12%稅率
- **計算示例**：若可課稅年收益為120,000元，首60萬免税，超出部分繳12%

### 免雙重徵稅協議

- 澳門已與**中國大陸、香港、葡萄牙**等國家和地區簽署免雙重徵稅協議書及稅務資料交換協議書

### 科技創新企業稅務優惠

- **《從事科技創新業務企業的稅務優惠制度》**：由財政局負責
- **工業政策範圍內稅務鼓勵**：由經濟及科技發展局負責
- 針對科研項目、專利申請、研發投入等可能有額外稅務減免

### 初創企業相關優惠

- 所得補充稅的免稅額對小型初創企業非常有利（首60萬完全免稅）
- 營業税基本豁免

## 五、澳門類似香港科技園的教育科技園區

### 澳門科技研發產業園（最新規劃，2025年）

澳門即將擁有自己的科技園區——**澳門科技研發產業園**，意義重大。

#### 核心定位（「三個平台」）

1. **內地優質科技企業出海陣地**：為內地科技企業走向國際市場提供服務
2. **海外前沿科技項目落戶服務站**：吸引海外科技項目和人才落戶
3. **本澳產學研基地**：打通高校科研與產業應用的通道

#### 選址規劃

| 地段 | 位置 | 用地面積 | 產業重點 |
|------|------|----------|----------|
| **偉龍馬路地段** | 氹仔東北側（近澳門國際機場、澳門科技大學） | 約8.3萬平方米 | **生物醫藥**（高端研發平台、訂製化廠房、產業服務配套） |
| **新城E1區西側地段** | 新城填海區 | 約17.7萬平方米 | **數字科技、集成電路、航天科技**（數據及算力中心、城市會客廳、公共服務大廳、濱水活動空間） |

#### 重點產業方向（四大產業）

1. **生物醫藥**（配合澳門科技大學等高校科研優勢）
2. **集成電路**（澳門已處於世界領先水平）
3. **數字科技**（數據、算力等數字經濟）
4. **航天科技**（國家戰略性新興產業）

#### 運營模式

- **特區政府牽頭 + 園區公司營運**
- 公共資本的澳門科技研發產業園公司負責建設、日常營運及招商
- 政府職能：整體定位、規劃、引入重大項目、提供專項政策支持

#### 與高校合作

- 與澳門大學、澳門科技大學等高水平高校深度合作
- 加快落實粵港澳大灣區全國高校區域技術轉移轉化中心建設
- 深化與江蘇及廣東兩地轉移轉化中心的合作

#### 與香港科技園比較

| 比較維度 | 香港科學園 | 澳門科技研發產業園（規劃中） |
|----------|-----------|--------------------------|
| **成立時間** | 2002年 | 2025年規劃（藍圖出爐，建設中） |
| **規模** | 更大（已運營多年） | 總用地面積約26萬平方米 |
| **主要產業** | 資訊科技、生物科技、綠色科技 | 生物醫藥、集成電路、數字科技、航天科技 |
| **孵化功能** | 成熟（技術孵化、風險投資、國際合作、人才培育） | 規劃中（科研—孵化—轉化—服務一體化） |
| **大學合作** | 與多所大學合作 | 主要與澳大、澳科大合作 |
| **特色** | 東西文化匯聚、國際化程度高 | 對葡語國家市場的獨特優勢 |
| **與大灣區互補** | 深港合作（河套地區） | 澳琴一體化（「澳門研發+橫琴轉化」） |

### 現有類似孵化器（替代方案）

| 孵化器 | 特色 | 是否聚焦教育科技 |
|--------|------|----------------|
| **澳大創新創業中心** | 已有教育領域孵化案例 | 部分涵蓋 |
| **澳門青年創業孵化中心** | 全方位創業支援，教育科技可申請 | 是 |
| **UMTEC** | 澳門大學產學研平台 | 可合作 |

## 總結：教育科技Startup在澳門的可行路徑

1. **初期資金**：申請青年創業援助計劃（最高30萬免息）
2. **科研及技術轉化**：申請科學技術發展基金的企業創新研發資助計劃
3. **孵化進駐**：澳大創新創業中心（師生/校友）或澳門青年創業孵化中心
4. **稅務環境**：所得補充稅首60萬免稅（12%税率），有利初創
5. **教育科技園區**：待澳門科技研發產業園建成後（預計生物醫藥、數字科技為主，教育科技需視規劃）
6. **市場拓展**：
   - 先立足澳門及大灣區（橫琴）
   - 再借香港國際平台出海
   - 利用澳門對葡語國家的獨特優勢
7. **進入香港**：沒有專門CEPA-type優惠，需自行按市場規則競争，但可利用香港國際金融、貿易平台
