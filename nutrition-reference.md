# 中國居民膳食指南 (2022) + 營養素參考攝入量

## 資料來源
- 中國居民膳食指南（2022）
- 中國營養學會
- 中國居民膳食營養素參考攝入量 (DRIs 2023)

---

## 平衡膳食八準則

### 準則一：食物多樣，合理搭配
- 每天12種以上食物，每週25種以上
- 穀類為主：200-300g/日（全穀物+雜豆50-150g）
- 薯類：50-100g/日

### 準則二：吃動平衡，健康體重
- 每週至少5天中等強度運動，累計150分鐘以上
- 每日6000步
- 每周2-3天高強度有氧運動 + 抗阻運動

### 準則三：多吃蔬果、奶類、全穀、大豆
| 食物 | 每日建議量 |
|------|-----------|
| 新鮮蔬菜 | ≥300g (深色蔬菜占1/2) |
| 新鮮水果 | 200-350g |
| 奶製品 | 相當於300ml以上液態奶 |
| 堅果 | 適量 |

### 準則四：適量吃魚、禽、蛋、瘦肉
| 食物 | 每日/每週建議量 |
|------|----------------|
| 魚禽蛋瘦肉總量 | 120-200g/日 |
| 魚 | 每周2次或300-500g |
| 蛋 | 300-350g/週 |
| 畜禽肉 | 300-500g/週 |

### 準則五：少鹽少油，控糖限酒
| 營養素 | 每日上限 |
|--------|----------|
| 食鹽 | ≤5g |
| 烹調油 | 25-30g |
| 添加糖 | ≤50g (最好<25g) |
| 反式脂肪酸 | ≤2g |
| 酒精 | ≤15g/日 |

### 準則六：規律進餐，足量飲水
| 性別 | 每日飲水量 |
|------|-----------|
| 男性 | 1700ml |
| 女性 | 1500ml |

## 蛋白質參考攝入量 (成年人)

基於中國居民膳食營養素參考攝入量：

### 根據活動水平

| 活動水平 | 蛋白質攝入量 (g/kg體重/日) |
|----------|---------------------------|
| 輕度活動 | 0.8-1.0 |
| 中度活動 | 1.0-1.2 |
| 重度活動 | 1.2-1.8 |

### 根據體重 (假設70kg成年人)

| 活動水平 | 每日蛋白質 (g) |
|----------|---------------|
| 輕度 | 56-70g |
| 中度 | 70-84g |
| 重度 | 84-126g |

### 減脂期間建議
- 建議攝入：1.2-1.6 g/kg體重/日
- 70kg成年人：84-112g蛋白質/日

## 碳水化合物參考

### 每日建議量
- 最低需求：100g/日 (防止酮症)
- 建議攝入：130g/日
- 供能占比：50%-65%

### 膳食纖維
- 建議攝入量：25-30g/日

## 減肥/減脂人士指引

### 熱量赤字建議
| 減重速度 | 每日熱量缺口 | 預期效果 |
|----------|-------------|----------|
| 溫和 | 300-500 kcal | 每月0.5kg |
| 中等 | 500-750 kcal | 每月1kg |
| 進取 | 750-1000 kcal | 每月1.5kg |

### 重要原則
1. **唔好低過BMR**：男性BMR約1500-1700kcal，女性約1200-1400kcal
2. **蛋白質要夠**：減脂期間保持1.2-1.6g/kg體重
3. **脂肪唔好太少**：至少20%總熱量
4. **慢慢減**：每週減0.5-1kg最安全
