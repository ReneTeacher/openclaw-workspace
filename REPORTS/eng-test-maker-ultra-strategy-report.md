## ⚠️ 執行摘要前必讀：最重要發現

### 發現一：Stripe 不支援澳門
Stripe 目前只支援 46 個國家，包括香港但**不包括澳門**。

| 方案 | 澳門支援 | 難度 |
|------|---------|------|
| Stripe HK | ⚠️ 需香港公司/身份 | 中 |
| **LemonSqueezy** | ✅ 最簡單 | 低 |
| **PayPal** | ✅ 可用 | 低 |
| FPS 直接轉帳 | ✅ 澳門可 | 極低 |

### 發現二：澳門沒有 DSE 考場
> **這是關鍵誤解。澳門沒有 DSE 考場。**
> 
> DSE（香港中學文憑考試）是香港制度，澳門學生如欲参加 DSE，**必須赴香港應試**。澳門大學有自己的「四校聯考」，並非 DSE 系統。

### 發現三：50-60% 佣金太高
業界慣例 15-25%，50-60% 會令你變成朋友嘅打工仔。

### 發現四：科大訊飛英文 AI 仍在筹备
科大訊飛喺澳門超過 50% 學校已落地，但**英文 AI 批改仍處於筹备階段**，呢個係你嘅時間窗口。

## 第五部分：朋友合作策略（佣金設計）

### 5.1 為什麼 50-60% 太高？

| 問題 | 後果 |
|------|------|
| 你承担技術、開發、客服、AI 成本 | 佣金太高蠶食利潤 |
| 房產經紀佣金約 1-2%，唔係 50% | 市場慣例唔支援 |
| 教育 SaaS 分銷佣金慣例 | 15-25% |

### 5.2 建議佣金結構

| 角色 | 職責 | 佣金比例 |
|------|------|---------|
| **引路人/顧問** | 介紹陳紅校長認識 | 一次性介紹費 HK$5,000-15,000 |
| **顧問委員會** | 提供市場建議 | 股份 0.5-2% 或月費顧問費 |
| **科大訊飛協調人** | 處理分銷合作細節 | 科大訊飛部分佣金（唔係你出） |

### 5.3 底線談判策略

- ✅ 總體佣金不超過 **25%**
- ✅ 朋友個人部分不超過 **15%**
- ✅ 必須有 **12個月上限**
- ✅ 設置**最低銷售門檻**
- ❌ 永遠唔答應 50-60% 固定分成
- ❌ 永遠唔答應**終身客戶綁定**

## 第六部分：用家旅程與 UX 改善

### 6.1 老師用家旅程痛點

```
[註冊/登入] → [選擇題庫] → [設定參數] → [AI生成] → [編輯題目] → [預覽] → [發布] → [查看報告]
```

| 步驟 | 痛點 | 放棄率 |
|------|------|--------|
| **編輯題目** | 操作複雜，不知道修改是否生效 | ~40% |
| **首次登入** | 帳戶/密碼問題 | ~35% |
| **設定參數** | 術語專業，選項太多 | ~25% |

### 6.2 學生用家旅程痛點

```
[收到連結] → [點擊打開] → [登入/輸入ID] → [答題] → [提交] → [查看分數]
```

| 步驟 | 痛點 | 放棄率 |
|------|------|--------|
| **答題介面** | 手機選項太小，容易誤按 | ~35% |
| **提交後不確定** | 不知道是否成功 | ~30% |
| **連結打不開** | 學校網絡封鎖外部連結 | ~20% |

### 6.3 UX 改善建議

**「三次點擊完成」原則：**
```
首頁只有：
✅ [一鍵生成試卷] （大字、醒目）
✅ [開啟上一次試卷]
✅ [幫助]
```

**澳門老師恐懼 Tech？降低學習曲線策略：**
- 提供「澳門中一英文月考」模板
- 「更多選項」折疊起來
- 每個步驟有小問號「？」即時解釋
- 首次使用有「導覽 Tour」功能

## 第八部分：增長策略

### 8.1 首 100 個付費用戶的階段性策略

**第一階段（0→10付費）：精準滲透**
- 借濠江英才關係做 Pilot
- 免費 6-8 週全功能試用
- 要求：每週反饋 + 介紹 1-2 位老師

**第二階段（10→50付費）：老師帶老師**
- 邀請同事 → 雙方獲得 1 個月 Premium
- 邀請 3 位同事 → 年度訂閱 8 折

**第三階段（50→100付費）：官方認可**
- 向 DSEDJ 申請工具認證
