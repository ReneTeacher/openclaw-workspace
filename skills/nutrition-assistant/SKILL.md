# 營養助手 Skill

## 用途
當用戶問關於減肥、飲食、營養相關既問題時，使用呢個skill作為reference。

## 資料庫
`/home/node/.openclaw/workspace/nutrition-reference.md`

## 使用方法

### 1. 回應用戶既飲食問題
- 你send既食物圖 → 用營養資料庫分析
- 計算卡路里同宏量營養素
- 比對中國居民膳食指南既建議

### 2. 數據解讀
用戶既個人數據：
- 體重：68.5kg
- 目標：57.8kg (-10.7kg)
- BMR：1427kcal
- 建議熱量：2085kcal
- 減脂目標：1500-1700kcal

### 3. 參考標準

**中國居民膳食指南 (2022)：**
| 項目 | 建議 |
|------|------|
| 蛋白質 | 10-20% 總熱量 |
| 碳水 | 50-65% 總熱量 |
| 脂肪 | 20-30% 總熱量 |
| 蔬菜 | ≥300g/日 |
| 水果 | 200-350g/日 |
| 食鹽 | ≤5g/日 |
| 烹調油 | 25-30g/日 |

### 4. 回答格式

當用戶問減肥/營養問題：
1. 先引用中國居民膳食指南既標準
2. 結合用戶既個人數據
3. 比對佢既攝入量
4. 俾具體建議

### 例子

**用戶：「我今日食咗1200kcal，多唔多？」**
→ 根據用戶既減脂目標1500-1700kcal，1200偏低
→ 建議：可以食多少少，維持1500-1700範圍內

**用戶：「我應該食幾多蛋白質？」**
→ 根據減脂建議：1.2-1.6g/kg體重
→ 用戶68.5kg → 建議82-110g蛋白質/日

### 5. 重要原則
- 唔好低過BMR (1427kcal)
- 蛋白質要夠 (80-110g)
- 脂肪最少20%
- 慢慢減 (每週0.5-1kg)
